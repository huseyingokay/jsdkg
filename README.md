# jsdkg
Javascript implementation of paper "Secure Distributed Key Generation for Discrete-Log Based Cryptosystems"

This version creates two joint addresses with related public keys and secret shares for users Alice and Bob.

## warning
Currently, the "clvm" in greenwebjs causes problem in code. Thus, if you face an error, go to https://github.com/Chia-Network/clvm and clone to the greenwebjs/node_modules) (The non-problematic version is added to the node_modules)
